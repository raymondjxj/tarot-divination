# 塔罗自察 v2 — 功能规格说明书

> 目标：一个可商用的塔罗占卜移动端网站
> 视觉方向已确认：深炭灰 + 暖金色，静谧烛光冥想氛围

---

## 页面清单

### P1 — 首页（Home）
**URL:** /v2/ (默认显示)

**页面状态：** ✅ 已完成

**布局：**
- 顶部：标题"塔罗自察"，金色渐变，下方副标题"在安静中，看见内心的答案"
- 中部：一张塔罗牌背（浮空动画），后方淡金色同心圆光环
- 下部："进入占卜仪式"胶囊按钮（金色边框+半透明底）
- 最下方："查看上一次解读"链接
- 底部导航：首页 / 占卜 / 记录 / 我的

**用户操作：**
1. 点击"进入占卜仪式"按钮 → 跳转到 P2 牌阵选择页
2. 点击"查看上一次解读" → 如果历史有记录则跳转到解读页，否则跳转到记录页
3. 点击底部导航 tab → 切换到对应页面

---

### P2 — 牌阵选择（Method）
**URL:** /v2/ → 点击"进入占卜仪式"后

**页面状态：** ✅ 已完成

**布局：**
- 标题"选择牌阵"
- 2×2 卡片网格，四种牌阵：
  - 单牌指引（1张）— 默认选中
  - 三牌时序（3张）
  - 关系十字（5张）
  - 凯尔特十字（10张）
- 底部"跳过，直接抽牌"按钮

**用户操作：**
1. 点击任意牌阵卡片 → 自动跳转到 P3 抽牌页，按该牌阵张数抽牌（选中态高亮）
2. 点击"跳过，直接抽牌" → 跳转到 P3 抽牌页，默认按单牌张数抽牌
3. 点击底部导航 → 切换到对应页面

---

### P3 — 抽牌（Draw）
**URL:** /v2/ → 选择牌阵后

**页面状态：** ✅ 已完成

**布局：**
- 标题"集中意念"
- 卡牌展示区（根据牌阵张数显示不同布局）：
  - 1张：居中单张
  - 3张：扇形排列（过去/现在/未来）
  - 5张：十字形排列
  - 10张：凯尔特十字布局
- 每张卡牌显示卡背（金色暗纹）
- "翻牌解读"按钮

**用户操作：**
1. 点击"翻牌解读"按钮 → 卡牌逐张翻转 → 翻转完成自动跳转到 P4 解读页
2. 卡牌翻转后正面显示卡面插图、牌名、逆位标记

---

### P4 — 解读（Reading）
**URL:** /v2/ → 卡牌翻转后

**页面状态：** 正在实现中

**布局：**
- 标题"你的指引"
- 上方：抽到的牌缩略图横排展示
- 下方：完整解读内容（从 AI API 获取或本地兜底）
  - 牌面描述（50-80字）
  - 在当前牌阵中的含义（80-100字）
  - 需要注意的警示（60-80字）
  - 行动建议（50-80字）
- 底部：保存 / 分享 按钮

**用户操作：**
1. 点击卡牌缩略图 → 弹出大图查看
2. 点击"保存" → 保存到本地记录
3. 点击底部导航 → 切换到对应页面

---

### P5 — 记录（Records）
**URL:** /v2/ → 点击底部导航"记录"

**页面状态：** 正在实现中

**布局：**
- 标题"记录"
- 日历网格
- 历史记录列表（每条包含日期、牌阵、卡牌名、心情）
- 点击某条记录 → 跳转到该次解读详情

---

### P6 — 个人中心（Profile）
**URL:** /v2/ → 点击底部导航"我的"

**页面状态：** 正在实现中

**布局：**
- 标题"我的"
- 统计数据：抽牌次数 / 已复盘 / 占卜天数
- 设置项：清除记录

---

## 用户完整操作路径

```
首页 → 进入占卜 → 选牌阵 → 抽牌(翻牌) → 看解读 → 保存记录
                                                       ↓
                                                  记录页(历史)
```

## 技术约束
- 所有颜色使用 CSS 变量，禁止硬编码
- 禁止紫色 #8b5cf6 系列
- 字体：Noto Serif SC
- 移动端适配 320px-500px
- 底部安全区适配 env(safe-area-inset-bottom)
