## 架构

单文件 HTML，零框架依赖。CSS + JS + 数据全部内联。

```
index.html          ← 完整应用（~170KB）
VPS 图片托管        ← 79 张 PNG 卡面
DeepSeek API 端点   ← AI 解读生成（Node.js + PM2）
nginx + Let's Encrypt ← HTTPS 反向代理
```
