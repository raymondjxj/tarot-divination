## 设计系统

- **主色调**：深黑紫 `#070312`，荧光紫 `#9b6dff`，暗金 `rgba(200,160,80,.x)`
- **中央仪式区**：多层同心圆天象轮（120s 自转）+ 塔罗牌面 SVG 金线绘制（全知之眼+曼荼罗+弯月+四角卷草）
- **字体**：Noto Serif SC（衬线），标题紫白渐变外发光
- **响应式**：全适配手机/平板/桌面，平板媒体查询收紧垂直间距

## 架构

单文件 HTML（~120KB），零框架依赖。

```
index.html          ← 完整应用（CSS + JS + 78张牌数据全部内联）
VPS 图片托管        ← 79 张卡面 PNG + 4 张方法卡图 + back.png
/var/www/sites/tarot/
DeepSeek API 端点   ← Node.js + PM2
nginx + Let's Encrypt ← HTTPS 反向代理
```
